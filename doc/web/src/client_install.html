Client Installation

Generally the EMF client is run from a Windows PC. Optionally, it can be installed and run on Linux, although there are some slight layout and behavioral differences.

  • Windows Client Installation using Installer
  • Manual Linux Installation

Windows Client Installation using Installer

  1. The EMF client runs best under Java 6. Before running the installer or the client, please download and install the latest Java Runtime Environment 6 (e.g., JRE 6u14)
  2. Download the EMF Installer Executable (emfinst.exe)
  3. Double click on emfinst.exe to run the installer.
    • First, the installer will expand the zipped compressed file onto your hard-disk. The default location is C:\emf-install. This is a very small installation of the installer software only, and does not require much disk space.
    • Once the expansion is complete, the installer program will start.
  4. In the EMF Client Installer window, fill in the fields. You may click "Browse" to help you choose any local directories.

    NOTE: the input and output file directories should be directories on the remote EMF server, not on your local client's machine. The installer will warn you if they don't exist on your client machine. You can ignore that warning and keep installing.

    Examples of filled-in fields are the following:

    • EMF Download URL: http://www.ie.unc.edu/cempd/projects/emf/beta_testing/install
    • Java Home Directory: C:\Program Files\Java\jre1.6.0_07
    • Input File Directory: /orchid/share
    • Output File Directory: /orchid/share
    • Client Home Directory: C:\Program Files\EMFClient
    • Server Address: http://emf.ie.unc.edu:8080/emf/services

  5. Click "Install".
  6. If you already have the client installed, you will be prompted about whether you wish to "Update" or "Reinstall". "Update" will download only changed files, and "Reinstall" will download all files. In most cases, you can "Update", but if things do not seem to be working properly you can try to "Reinstall".
  7. The installer will provide feedback on the files it is downloading. The latest list of installation files is available from files.txt
  8. When the installation is finished, click "Done".

Note: A file containing the preferences (i.e. stored settings) for the installer called EMFUserInstallPrefs.ini is stored in your user home directory (usually C:\Documents and Settings\<your_user_name>), and a file containing the EMF preferences called EMFPrefs.txt is in the same directory.
